# react_webpack
React Webpack Basic Setup

### Install, build, run
npm install  
npm run start  
npm run build

this set up requires babel-loader7.x in order to work with babel-core6.x

### Dependencies
babel-core  
babel-loader  
babel-preset-env  
babel-preset-react  
html-webpack-plugin  
webpack  
webpack-cli  
webpack-dev-server  
react  
react-dom
